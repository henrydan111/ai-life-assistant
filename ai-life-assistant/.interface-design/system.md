# Design System

## Direction

Personality: Quiet command desk for a personal life assistant.
Foundation: Light, tactile, calendar-aware, more human than admin.
Signature: A time-horizon ribbon connecting capture, near-term planning, and later horizons.

## Tokens

### Color

--background: #eef3ef
--surface: #fffdfa
--ink: #17211d
--muted: #64736c
--accent: #217766
--warm-accent: #d29b57
--soft-line: #d8e1da

### Type

Display: Iowan Old Style / Georgia fallback, used only for page and now headings.
Body: Avenir Next / Helvetica Neue fallback.
Utility: SF Mono for compact labels and brand mark.

### Layout

Base spacing: 4px grid.
Common gaps: 8, 12, 16, 18, 24, 32.
Panels: 18px radius, low elevation, warm white surface.
Controls: 40px desktop minimum, 44px mobile minimum.

## Patterns

Primary action: green gradient, white text, restrained shadow.
Calendar horizon: top color strip keyed by immediacy.
Now panel: tinted surface with a left time strip.
Background: subtle calendar grid over a soft field, never blank flat white.

## Guardrails

Keep the interface work-focused. Do not add hero marketing sections.
Spend visual boldness on the time-horizon ribbon and warm typography.
Avoid purple-first palettes, generic blue SaaS styling, and identical white cards.
